# NoteDraftForge UX Wireframe (React + Tailwind)

MVP-only wireframe screens based on provided references.

## Included screens
1. WorkListDesktop
2. WorkListMobile
3. PieceVisualizationDesktop
4. PieceVisualizationMobile
5. PieceEditingDesktop
6. PieceEditingMobile
7. AnnotationModalDesktop
8. AnnotationModalMobileBottomSheet

## Scope constraints applied
- poem/text UX only
- no song creation/edit/view flows
- no import/export flows
- visualization and editing mode only
- visualization default representation
- layer toggles only in visualization
- annotation actions visualized only in visualization + snapshot-present context
- editing mode has placeholder editor area only
- fake navigation + mock data only
- no persistence
